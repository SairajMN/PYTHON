# Learning Python with Micro Projects

This repository contains various small Python projects to practice and demonstrate fundamental programming concepts.

## Projects

### Basic Programs
- **sumofn.py**: Calculates the sum of first n numbers using a for loop
- **stringf.py**: Demonstrates basic string functions and operations
- **fibonacci.py**: Checks if a given number is part of the Fibonacci sequence
- **calculator.py**: Command-line calculator performing addition, subtraction, multiplication, and division
- **fortune.py**: Fortune teller program that gives random answers like a magic 8-ball

### Data Structures and Algorithms
- **selections.py**: Implements selection sort algorithm
- **sequentials.py**: Implements sequential (linear) search
- **abcSTACK.py**: Demonstrates stack operations (push/pop)
- **stack.py**: Additional stack implementation
- **dictionaries.py**: Examples of dictionary usage and operations

### Advanced Examples
- **conway.py**: Implementation of Conway's Game of Life cellular automaton
- **students.py**: Student performance analyzer with features to add students, calculate averages, find toppers, sort by performance, and search records
- **bdaydb.py**: Birthday database using dictionaries with methods to add, query, and display birthdays
- **regularexp.py**: Demonstrates regular expressions for pattern matching

### Other Utilities
- **RPS.py**: Rock Paper Scissors game
- **guess.py**: Number guessing game
- **primenum.py**: Prime number checker/generator
- **quadratic.py**: Solves quadratic equations
- **collatz.py**: Demonstrates the Collatz conjecture
- **mtable.py**: Multiplication table generator
- **zigzag.py**: Zigzag pattern generator
- **listsofcats.py**: Working with lists of categorical data
- **filef.py**: File handling operations
- **ex.py**: Example spam function
- **bobfun.py**: Simple greeting functions
- **fun.py**: Additional greeting functions
- **loops.py**: Loop examples
- **list.py**: List operations
- **spam12e.py**: Spam example
- **bobfun.py**: Bob's fun functions
- **sequentials.py**: Sequential operations
- **test.txt**: Test file for file operations

### Running the Programs
Each Python file can be run independently using:
```bash
python filename.py
```

Most programs are interactive and will prompt for user input.
